# 7/3，Villares de Orbigo
- 享受>品味>鑑賞>創作，這可能是一個藝術家有能力創作藝術品，從最開始到開始創作的過程
- 只有一開始似乎會比較孤單，思念家人朋友，久了其實無感
- 還記得以前剛上班時，享受清閒，或現在偶爾在camino的時候會有一種人生至此，夫復何求的感覺
- 那瓦爾寶典裡面有一個論點
	- 人生的履歷就是承受的苦難，我感到很有道理，更精也許是「承受苦難的能力」、「承受什麼類別、長度、程度樣的苦難而可泰然自若，處理得宜的能力，包含外在的技能與內在的心理能力」
		- 例如一個人反復重病或被車撞，這應該不是他所指的承受的苦難，他指的是對此泰然自若、處理得宜的能力 
	- 可以重新理解10幾歲時，我給自己的信中，為什麼希望承受七苦八難，因為我渴望成長，我知道承受苦難並且戰勝他是成長的好方法
		- 一個簡單的例子來說，幼年可能不能吃到想吃食物的而不愉快，甚至大哭大鬧，但成年之後很少有人、有那種時機會因此難過到流淚
		- 成年不再受感官體驗過度左右心情，承受苦難的能力上升，這就是簡易版本的成長，這個成長就是人生的履歷
		- 再進一步來說，可以在大部分人認為高壓的環境求學工作、可以創業、可以創業失敗後再創業、可以做到前述又兼顧家庭與友情、可以飽經風霜又保持真誠、可以受到傷害後保護未來可能被傷害的人而不是認為他要跟自己經歷一樣的苦、可以被愛人傷害仍相信愛情等等，就是困難版本的成長
	- 同時在真的稍微經歷一點苦難後，也可以理解大學或出社會的我為什麼覺得承受七苦八難的這個願望太過年輕，因為有些苦難沒有可以反復試錯的機會，一次就足以需要多年休養或一覺不振，也許他在世俗價值的功名利祿一樣優秀傑出，但他可能不願意再保持真誠，而不保持真誠是痛苦的，最後會連自己一起欺騙的
		- 所以夢幻般地成長路徑是「連續承受多個在自己承受極限的苦難，每個苦難間隔恰當的時間，或是偶遇貴人相助」，當然，夢幻之所以夢幻，是因為無法達成
			- 例如自己所能承受的極限有時是連自己也不知道，只有上帝知道的；當然我們更不可能完全地控制苦難來臨的時間
			- 也許這個苦難稱之為挑戰更加適合?
	- 很遺憾地，我們的人生也會遇到毫無成長的人，這種人理應讓人感到厭惡，但有時我難以對這種人感到真正的厭惡，我會不自覺得做一個假想實驗
		- 假設我失去記憶，我的基因換成他的基因，我投胎成他，把他的胚胎換成我，其餘一切照舊，長大後我會比較像我還是他?
		- 假設我失去記憶，我的基因維持，其餘照舊，長大後我會比較像我還是他?
		- 假設我保有目前記憶，其餘照舊，長大後我會比較像我還是他?
		- 只有保有記憶的情況我認為我會比較像我
			- 人能改變的事情同時比想像中多、也比想像中少
				- 我完全相信一個人由習得性無助的人格轉為成長性人格人生會徹底不同
				- 但如果真的完全比想像中多，不管怎麼樣前面問題的答案我應該都比較像我才對，我時常為此黯然神傷
- 到下午兩三點的時候幾乎都只有我一個人自己在，覺得有點意外，大家都那麼早起都走那麼快的嘛!?
	- 但也沒有什麼不好我就是按照自己的節奏做自己想做的事情
- 昨天和兩位英國年長女士吃飯，還有兩位中國人坐比較旁邊，不太參與
	- 英國女生45、55，都沒有結婚，55女生說結果一次離婚了，絕對不來第二次婚姻，45歲倫敦女生已經旅遊3年，55歲英國南方女生經營旅館多年（工作9個月，休息1.5個月，再工作1.5個月，突然發現旅館業沒有所謂的假日，頂多是淡旺季）第一次出來長途旅遊
	- 後來跟兩英國女生聊天得知，老闆娘是先賣單雙人房再賣宿舍房，老闆娘走前叫我門要關起說避免噪音，但其實不是吵鬧的問題，是為了不想被其他人看到宿舍房還很多空位，我覺得在還算合理的求生手段範圍內，不到惡劣的程度，因為這邊經營真的感覺很不容易，不知道其他人怎麼看
	- 中國男生西班牙語超強，感覺是在西班牙工作；中國女生好像沒有說中文以外的話。
		- 也許我應該邀請他們一起加入聊天，他們不和我們一起聊天也許是以為我們一開始就認識不想打擾，或者說至少試過
- 終於不拉肚子了
- 整個宿舍房間裡面只有我一個人，好舒服!今天還是禮拜五，感到有點奇怪，這個位置民宿這們難以經營嗎

- 在Astorga有高第的建築，
	- 第一眼就讓我震驚，於是我折返了幾百公尺想進場參觀

	- 這時候我竟然想放棄參觀，繼續走路避免晚到庇護所
	- 還好我又堅持下去回去入口處買票，明明人很少確排了兩組人，
	- 遇到Peter！

# 7/5，El Acebo de San Miguel
- 在聽哲學科普影片時，了解到他心問題的「顏色顛倒實驗」
	- 這個假想實驗是說如有一個人類眼睛中看到的A顏色其實是別人眼中的B顏色，他或其他人可能一輩子都無法發現，作為人類永遠無法完全了解彼此的論點之一。
		- 例如有個小孩從小看到的紅色是別人的綠色、綠色則是紅色；家長或老師在教育顏色分辨時，唯二能使用的媒介是語言和視覺，語言大家共通，而且視覺也會假設共通，如果指著西瓜內部說這是紅色，小孩明明看的是其他人眼中的綠色，但還是會將語言中的紅色與其他人眼中的綠色連結、反之語言中的綠色連結為紅色，而其他人乃至小孩自己是無法發現的
	- 這個論點看似好像很牢不可破但我認為有一個很大的問題，就是成立的前提須要滿足以下
		- 「雙方產生類似顏色錯置的不理解時，內心、外表或物理場景要毫無破綻」
			- 例如痛苦不痛苦、一個物體沉重不沉重、一個位置很高會很矮，這種概念他會產生特定的情緒或物理狀況，這種情況是沒辦法產生剛剛說的顏色不理解的情況
			- 再來是不理解的標的本身要有封閉性，才能夠被徹底隱瞞。
				- 若小孩看到的是，「其他人的紅色對自己是綠色，其他顏色與常人無異」，這樣小孩就會認為西瓜內外都是綠色的，在語言共通的前提下，馬上就會被其他人發現小孩的視覺和自己不是共通的，可以再透過更多的說明、實驗來化解這個不理解
	- 所以我認為「人與人必然有可以互相理解以及永遠無法互相理解的部分，不需要過度強調其中一部份而抗拒理解或認為一定可以理解對方，或因此產生什麼額外的情緒」
- 昨天遇到一個荷蘭刺青師我們約了今天一起進餐
	- 他父母離異，16歲輟學他堅持找到自己的熱愛
	- 他提醒我人的語言會被宇宙回應，被自我驗證、我們應該更謹慎說話
	- 他是真正的自己人生的英雄，母親早亡、父母離異、16歲離開學校、學會處理困境，他找到自己熱愛，爭取該做的事請，他現在在做藝術教學加上刺青師的工作，他克服了許多困難，他也承認人生有許多困難的地方
	- 他真的讓我印象很深刻，camino遇到這樣一個人就值得了，我極度真心地祝福他與他的女兒一切都好，他叫做Dylan（笛龍） 他的女兒是畢莉，他請我寫下中文的翻譯，說會保存好，我很感激，我很少被如此重視，我真心希望他與他的女兒一切過得比我想像中還要好
	- 刺青師有一個女兒，但他沒有結婚，但他聽到我說要給小孩一個家庭的論點沒有生氣，沒有反駁，反而是贊同
	- 我們聊了很多，他勸我不要變成賺錢機器，聽從自己心裡的聲音，在我們最後的擁抱他似乎怕我做不到哈哈哈，又再提醒了我一次，我非常感謝，跟他比我的確是膽小多了
	- 他推薦我蒙古音樂，非常特別
- 夕陽時我一直很想拍照，好像想留住這美景，今天真的是很棒的一天，很棒的旅宿，很棒的景色，遇到很棒的人
- 原來pan（印度裔加拿大人）是微軟員工，我覺得他很厲害，說話條理清晰而富有同理心與進取心態

- 想要做一個角色個性元素表
	- 世界上真實腳色或是虛構腳色泛用
	- 每個被創作出來的角色以不同的比例包含著這些元素
	- 給予恰當的世界觀以及模糊的主線劇情讓角色自由人在世界裡面發揮
	- 讀者降低角色為劇情服務的刻意感覺
- 何謂好作品?
	- 跨越時間與空間
		- 不同時空的人都有巨大啟發、震撼、悲愴、釋懷、狂喜、絕望、興奮、噁心、反思、慌張等等強烈感受
		- 並深深在讀者身上雕刻刻痕，影響他未來的人生軌跡，他在看完與看後這本書，本質上已經是兩個人，他的人生有沒有看過這本書，會發展出兩個不完全一樣的人生
	- 怎麼做到?
		- 自行回答、引用回答、提出或至少不迴避深刻的哲學問題
		- 真實感極強
			- 每一個一細節都讓讀者覺得很可能發生；每一個還沒閱讀的劇情(未來)都是無數可能結果的疊加態，要讓讀者覺得坍縮成這一個結果夠合理，要是在真實地球發生也很可能如此。
			- 為了要真實感強
				- 邏輯要嚴密、前後劇情不衝突、各角色人格特質原則上不輕易改變、改變務必合理
				- 作者閱歷豐富或是考究深入，在提到某種較為專業的事情時必須深入了解，
					- 例如建構一個小提琴家可以知道以下內容，能精確地知道越多，而且自然地流露於作品，一般讀者會覺得合理、讀者是相關行業的會覺得愉悅
						- 拉小提琴什麼部位容易有什麼樣的職業傷病、怎麼避免
						- 小提琴的價格、最好和最入門的小提琴是什麼
						- 不同等級的演奏者的出場費、出場機會、能出場的場合是什麼
						- 什麼樣的曲目是演奏者公認的難曲、出師曲、入門曲
						- 不同國家對於小提琴演奏的喜愛、培養能力，哪一個國家出產最多厲害的演奏者
						- 小提琴保養的細節
						- 不同等級演奏者之間相異與相同的地方
						- 以上粗略列舉，回台灣再整理，應該可以做成一個各行業的思索表，建構一個角色時有些是必然要思索的，不然角色注定過於單薄

# 7/9, Sarria 
- 上午去收衣服的時候門很難開還以為被鎖在外面，身上沒帶任何東西在那邊開了很久，還好後來用力一點後門開了，不熟悉的環境即使一點小地方都可能造成麻煩
- 從昨天開始我感覺開始蠻難過的，覺得這個旅程要結束
- 有點後悔走Samos這條分岔，挺遠的
 - 文明的瓶頸
	- 如果一個文明沒有辦法發展出更加便捷儲存記憶跟傳遞記憶的方式，就會讓科技發展的水平永遠停滯在某個程度
		- 智力
		- 壽命
	- 因為發展新的科技總是需要先了解整個前因後果，也就是以前所發展出來的知識才可以繼續，尤其數學最讓人有這感覺
	- 這些成本就是時間以及一個人的智力
		- 未必每個人的都有辦法了解夠高深的知識，了解之後未必有辦法拓展新的，所以需要足夠基數的人類去篩檢出智力足夠的人，也需要這個人壽不能太短（特指在智力水平仍高時的壽命）
		- 至少會有智力跟壽命這兩個成本，但是只靠這兩個成本總是會有用盡的時候，因為人類的演化並不一定會往智力越來越聰明或壽命越長去演化，而在這個物種智力以及壽命所能達到了瓶頸的時候也就是這個文明的科學達到了瓶頸
	- 如果在這個瓶頸之前沒有辦法發展出儲存傳遞知識的便捷裝置的話這個我們的科技進步將極其緩慢
- 大問題節目裡面提到了幾個為什麼沒有外星人或者說反駁費米悖論的幾個方案，那我想到還有一個是我小時候做的夢我姑且稱他為培養皿假說，和節目內的創世者方案類似
	- 夢境是有許多水晶球整齊劃一的排在木板格子內
	- 水晶球裡面有一個黑點在蠕動，培養水晶球的是一群幼稚園學生，類似他們生物課
	- 而黑點是宇宙，因為宇宙會不斷向外膨脹，而我們人類就在黑點裡面
- 今天的住宿非常震驚我以為我只是一張單人房，結果竟然是整層公寓只要47歐元!!太扯了，然後開門的方式也很特別，是他會傳送一個連結，連結打開之後可以直接用網頁裡面給按鈕開門，浪費我很多時間去找，因為我以為那個連結是釣魚連結

	- 這和評論婚姻生子也是一樣的，過度支持「有/沒有婚姻必然比較好，且另一方必然比較差」或「生/不生必然比較好，且另一方必然比較差」的人，恐怕也是出於恐懼，這種恐懼可能來自於自己沒有思考就行動、可能來自於自己已經發現自己錯了、可能來自於觀察別人覺得另一條路好像也不差，真的走在自己路上圓滿自得的人，似乎不會有想跟別人爭論這些選擇的、想要別人肯定自己的選擇是對的的動力，大部分是分享，而非談論到這話題就試圖辯論。
	- 也許可以觀察自己或他人，是否持有「特別想要與人辯論且希望自己贏的話題」，代表自己在這個題目上有所恐懼、尚有不成熟

# 7/17，Taipei
- 還記得一開始飛機陸地滑行，總覺得很久，很不耐煩，回來上海時已經沒有這種感覺了
- 我回憶起第一次遇到Dylan跟我說的語言的重要性，講出來的話或是對自己的想法，會成為自證預言，Dylan的用詞是universal will listen
	- 例如我似乎不斷告訴自己自己不喜歡人多的地方，慢慢成了自證預言，我原本真的是那麼不喜歡人多的地方嗎?
		- 人多會產生焦躁感，但是最好的情況應該是任何情況都泰然自若才對，一些無法改變的事情，不用過度分辨自己的喜歡或不喜歡，會讓自己對環境的適應性下降
	- 例如我以前覺得外表不重要，重要的是頭腦內的東西（歐巴馬演講或是beauty but skin deep)，但後來發現兩件事情讓我反思
		- 第一個是這個想法成為自證預言，我變成有點一個不修邊幅、不善穿搭的人
			- 我可以選擇要不要做這件事情，但不應該不培養這種能力
		- 第二個是審美是一個人漫長累積的表現，乾淨整潔是基本，穿搭保養是進階，它是一個人表現內在最容易的方式，一個人的外觀印象就像是一個人的傅立葉函數一樣，是過去所有時間的疊加，過去所有的細小選擇，讓自己現在的外觀常態是某種狀態、某種風格或是根本無法自理與審美，不該被輕忽
