# 6/13,PVG
- 中午同事原本想和我一起分享水餃，可惜我下午請假中午沒在公司吃了，同事送給我一顆蘋果，因其有平安的寓意我覺得很開心
- 騎車
	- 原本一直很介意的出發時天氣，一直在想騎到捷運順不順的事情，結果今天真的下雨，倒也覺得不是那麼重要了，反而是那些擔憂造成的壓力還比較大
	- 等高鐵車期間又在看著氣象，Sjpp和Roncesvalles前後都是一週的晴天，剛好抵達那天被我遇到雨天，想說怎麼那麼倒楣
		- 結過更倒楣的還在後面
- 高鐵
	- 發生一些插曲，我以為自己的外幣提款卡有信用卡功能但一直刷不過
	- 發現後開始思考趕回住宿拿卡
	- 後來又轉念，買紙本票就可以了
	- 上車時又想到，手機google pay就可以了
	- 因為害怕延誤後續行程，人在著急時腦子實在是不好使
	- 看到高鐵顯示14：50到達台北時，突然有一種幻想是回台北老家過一個月不是去camino
- 機捷
	- 桃園高鐵捷運往領航站開的極慢，這兩站離得很近，但高架又轉彎多造成，我內心產生煩躁，而且想上廁所
	- 搭捷運途中有種想把所有搭機流程快點跑完的急躁感，就想有時想快點把整個人生過完一樣
- 報到/託運
	- 軟包行李(登山包)一定要走人工，硬殼才能自助，另一個選擇就是不帶登山杖背著(這是我到現場才知道的事情，可以更早知道)
	- 在桃機弄自動行李託運不會掃描護照，遇到陌生人中年女士說了句對不起後，非常友善地教我怎麼掃描
	- 行李只能掛到上海浦東，我在那邊要重新領回行李、行李托運，重新排隊跟找登機門
		- 上面寫著23：20登機，23：50起飛，19：35抵達，應該是有足夠的時間，不過登機證上的D87登機門有點嚇到我，一個登機門如果跟桃機一樣遠，應該除了走路外有其他交通方式
		- 結果浦東規劃很好、很大，但不難找，D87又是一個不必要的憂慮
- 候機
	- 整個忙完，報到，托運，走路坐下來時是16：03，登機時間是17：00，起飛時間是17：30
		- 比我預計中晚33分鐘，主要消耗在軟包行李不能自助托運、找登機門有一點小繞路、吃兩顆茶葉蛋裝水
	- 右腳正前方的肌腱還是有點疼痛

- 搭機至PVG(上海浦東機場)
	- 華航空服小姐特意向兩個人報了姓名，對其中一個說差一個人就可以起飛，問另一個問要不要換座位
		- 這段讓我覺得很奇怪，這兩個人跟這位是什麼關係?既不知道姓名，但是又需要特別關心一下他們兩位，上級?
	- 我打算先不吃暈機藥來比較差異，到上海兩小時應該不至於太嚴重
    - 後來結果是吃飯時噎著胸口有點想吐，其他時候幾乎無感
	- 空服員問「需要加點熱咖啡嗎」對我來說有兩種意涵
		- 一種是熱咖啡要加錢來點餐，一種是要不要「加一點」
		- 我搞不清楚是哪一種所以都說不用，但咖啡應該是不用錢的，只有酒類要
	- 旁邊的中年中國女生垃圾掉了他看起來沒有要撿，我猶豫了一下決定日行一善撿了起來
	- 按壓右腳踝前方偏外側並轉動腳踝時很疼痛，我感到很憂心
- PVG機場
	- 在pvg機場的國際線空服員很親切
	- 領行李時我幫轉彎卡住行李車的人抬了一下
	- 一小部分中國人給我的感覺比較會插隊、往前擠，你不佔好他就會想搶
	- 這裡的警察感覺相當威嚴，出關時警察吼了別人一下不知道是什麼事情
	- 拿了台胞證就是走中國公民的人工通道，以前它對我只是一個去中國的證件，過通道才搞清楚什麼是台胞的意義
	- 門口的警衛指出第二航廈是出去後門二樓這個看似很模糊簡短不耐煩，但實際走了之後覺得相當精準簡潔，只是距離太長心中還是會怕怕的
	- pvg很大，走錯回頭大概要很久
	- 中國安檢非常嚴格，相對台灣來說，航廈之間又安檢，出關安檢嚴格很多，台灣有種迫於人很多趕快弄一弄的感覺，東西也不用分散很多格子放
	- 從19：15下飛機，到D87登機口坐下來，已來到20：45
	- 在pvg機場即使連了wifi也要翻牆，沒有想到這點，我手機裡並沒有能在中國用的app，等於20：45過後沒網路用
	- 現在電量83，有點慶幸在飛來pvg的路上有充電
	- 發現bilibili可以用，不錯
		- bilibili 聽到莫愁鄉(作者亞細亞的曠世奇才)，非常感動，回來要分析好歌為什麼好
- 很驚嘆於我這樣一天可以打這麼多字

# 6/14,FRA
- 搭機至FRA(德國法蘭克福)
	- 搭長途飛機來FRA的過程我很驚奇地發現了幾件事情，
		- 飛機的噪音對我來說影響不大，甚至有種安全感，讓我昏昏欲睡
		- 飛機上若睡覺或吃暈機藥的影響，其實不太想上上小號
		- 靠窗座位可以放頭真的太棒了
	- 我在前往FRA的路上睡得很舒服
	- 大部分人在開始睡眠後座位會往後調整
	- 靠窗座位站直的話即使是170的身高在德國漢莎也會幾乎撞到頭
	- 晚上2350啟航，0100左右硬是給了我們一份飛機餐,感覺挺詭異的，誰這個時間有吃飯的習慣或需求，半夢半醒被挖起來吃飯感覺很奇妙、半夢半醒吃完、半夢半醒又睡回去
	- 很慶幸有穿發熱衣,飛機上的小毯子蓋腿剛剛好
	- 玩著飛機座位上的小遊戲黑白棋，原本輸了好幾場，但我總想著再試試看，靜下心來嘗試，最後贏了，並且可以一直贏，我又學會一種棋牌小遊戲，感覺很不錯
	- 隔壁的中國女生玩麻將好強阿，勝率有9成了吧

	- 取消後經歷了許多事情，整個過程挺慌亂的，因為我沒有收到mail通知

# 6/15,BIQ
- 飯店起床
	- 睡不太好
		- 頸椎枕覺得舒服卻睡不好應該是時差問題
	- 上午我0430就開始在等0500的接駁車，一直拉肚子，已經開始拉水了
	- 有個要去瑞士的女生友善問我幾點的飛機跟要不要喝咖啡，我回答I know好像有點嚇到他哈哈哈哈，才反應過來這真的是很不禮貌。我想講的是「剛剛櫃檯有跟我說過，但我不用咖啡（空腹怕肚子痛）」,我應該要說的是Thank you, I'm good.
	- 在車上我詢問是整個歐洲車都那麼快還是只有德國，他們回我只有德國，真是個好國家，道路上沒車限速有意義嗎，不如把那些處分都加在未保持車距、惡意超車等危險駕駛

- 登機報到
	- checkin一波三折
	- 若有換機重新訂票（rebooking)
		- 要用自助機重新刷護照就取得換機證
		- 然後再持換機證去相對應的人工櫃檯（飛行公司不是原公司！）取得登記證
		- 這些應該6/14號就可以先做的事情，可惜漢莎的櫃檯沒提醒我，我其實在6/14應該就可以取得登記證了!而不是放在緊迫的6/15做這些事情
	- 取得新登機證後(照理說有收到mail上面應該會有)，得知不是原本的漢莎航空了，還好漢莎很清楚的告訴我在第二航廈，沒有在第一航廈浪費時間，中間有印度臉孔的健壯男生很友善
	- 到了第二航廈找不到櫃檯，航廈很大，舉不所見不知櫃台在哪，慌亂地晃了晃後問了清潔員，得知在相反方向
	- 看到排隊人潮心涼了，非常長
	- 站務人員過來引導排隊人潮說說可以用自助機，原本很高興，結果自助機沒法用(因為我是換機)，心又涼了，而且一開始排的一小段隊伍要重新排
	- 排隊時心裡一直在想怎麼那麼慢，最後發現就是某幾個困難客戶慢而已，其實他們效率很不錯，態度也很友善，那位幫我服務的男生態度也很好
	- 終於排到也順利拿到登記證時真的是鬆了半口氣
	- 趕快衝去安檢，安檢人員有位女生態度不知道在差什麼東西的，it's ok口音很重，跟其他安檢人員或地勤人員截然不同，不過耽誤時間不到1分鐘吧還行
	- 過完安檢再趕快衝去登機門，抵達登機門時間時間是06:19（登機時間06：35~06：45非常短），實在是非常刺激，距離登機門關閉時間只有24分鐘!
		- 還好從旅店過去的車程只用了google一半的時間（原預計28分鐘因為德國車速很快僅開15分鐘）
	- 感覺到人遇到不好的事情，當下或回憶產生的負面情緒會加以放大，記憶較久較深；而同等級的友善感恩部分則需特意牢記。這恐怕是某種人類的本能避免危險的古老機制

- 飛往CDG
	- 到了登機門後，仔細閱讀機票，發現CDG>BIQ的飛機很快就起飛，但沒有標示登機門，推測是不用過安檢不然應該來不及，上網查詢航班知道是G40登機門，這時我還在感嘆於自己的機智，知道要在上飛機前先查好登機門節省時間
	- 後來的確是不用過安檢，但是登機門在飛行前又再次改變了!所以我在FRA上飛機前查的登機門是錯的，那時在錯誤的登機門那邊，我用上飛機前的截圖予旁邊的女士確認她的目的地是否一樣，她回復我後說「It seems like one of us need to run very fast right now, pretty excited」
	- 哈哈結果是我要run very fast，從電子看板看到登機門G40變成G26，差點跑死，離登機時間差4分鐘
	- 不過這是臨時被改登機門，並不是我查詢到錯誤的登機門，連機組人員也相當晚抵達，班機也因此延誤飛行，後面其實還是要等，前面的匆忙算是沒有必要

	- 今天到SJPP應該可以先弄朝聖者護照
	- CDG機場連不上booking和agoda讓我有點小煩躁，沒有辦法先訂SJPP的旅店
