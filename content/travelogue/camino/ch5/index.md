# 行李寄送與腳踏車租用服務
因為這兩個服務我並沒有實際使用，所以只能做簡單介紹
# 行李寄送
- 大部分旅館都有提供行李寄送服務
	- 只要在前一天晚上或當天清晨，跟櫃台領取資料卡
	- 在資料卡寫上下一個住宿地點，再將資料卡固定於行李上
	- 最後集中放置在那間旅店規定的地方
	- 就完成寄件流程
- 關於行李寄送，建議第一天攀登庇里牛斯山時若有下雨是推薦寄送的
	- 當天細雨和陰天交錯，一同出發的一位美國老太太可能因為太濕滑腳踝嚴重扭傷，需要救護車來接送
	- 即使是小雨，的確有小部分斜坡路段非常難立足，負重情況下走那種路，可能某些肌群的支撐、或鞋子的止滑能力是不足夠的
# 腳踏車租借
- 那時我曾經懷疑自己到後面會很精疲力竭，想要在中途的平原區騎車，那時有查閱相關網站
- 整個租借看起來已相當完善，包含車種、保險、自訂租借起點終點等
- 介紹與相關租借平台:
	- [Camino ways介紹](https://caminoways.com/camino-bike-rental#h-how-camino-bike-rental-works)
	- [Bikeleon](https://bikeleon.com)
	- [Cycling the Camino](https://cyclingthecamino.com/)
- 另外有些路段其實騎車比走路還累，尤其剛開始的山區丘陵，只有平原區最適合騎車
# 預算
- 我這邊沒有詳細的清單，當時不含機票預算抓15萬台幣，總結算是含機票是花11萬台幣，大概可以分成通勤、餐宿、其他3個部分
- 通勤
	- 機票3.4萬台幣
	- 台灣高鐵、當地列車大概0.3萬
- 餐宿
	- 餐飲預算可抓1200歐
		- 每日飲食20-30歐，抓900歐
		- 大城市有大餐可吃時會花50-60歐，抓300歐
	- 住宿預算可抓800歐
		- 住宿大部分15-20歐，抓20天，400歐
		- 大城市有單人房時到40歐，抓10天，400歐
- 其他
	- 耗材:例如補充醫藥品、盥洗用品
	- 參觀:各處較具規模的教堂、博物館等會需要收入場費或背包置物箱費
	- 寄送:沿途旅館行李寄送服務費用
	- 備用
- 預算總結
	- 以2025的經驗來說，15萬台幣含機票(需轉機)算是綽綽有餘了
	- 那裡想花錢也沒地方給我們花，買個可樂面對自己的內心、活在當下，會發現必要的實在是很少，一個後背包就是一切所需
